# Synapse v0.1.0

First public OSS preview release of Synapse as an Agentic Wiki and cognitive state layer for AI agents.

## Highlights

- Universal SDK loop in Python and TypeScript (`observe -> synthesize -> curate -> execute`).
- OpenClaw-first onboarding with canonical 5-minute quickstart and runtime connector path.
- Human curation web workspace with core-first moderation flow and production-like UX.
- MCP-native retrieval runtime for approved knowledge and task context.
- Agentic Task Core (`/v1/tasks*`) with SDK helpers and OpenClaw tool wiring.
- Day-0 memory bootstrap flow (`backfill` + attach-time bootstrap presets).
- Regression/quality gates:
  - synthesis/gatekeeper/retrieval deterministic evaluators,
  - web Playwright e2e,
  - cookbook golden snapshots,
  - docs canonical path lint for OpenClaw onboarding.

## Notable In This Cut

- TypeScript OpenClaw parity:
  - `Synapse.attach(runtime, { integration: "openclaw" })` now wires hooks and tools by default.
  - Auto search fallback via SDK retrieval (`search_wiki`) when no custom callback is provided.
  - Signed provenance chain on `propose_to_wiki` (HMAC when secret configured, digest fallback).
- Cookbook stability contract:
  - Added golden output snapshots and CI snapshot checker for all cookbook scenarios.
- Product polish:
  - Core wiki UI theming pass and hero diagram refinement.
  - OpenClaw docs now guarded by canonical reference lint in CI.
